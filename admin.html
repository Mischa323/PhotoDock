API Reference

Authentication

All API endpoints require an API key. Pass it in one of two ways:

Header x-api-key: YOUR_KEY
Query https://your-server/api/slideshow/current?key=YOUR_KEY

Endpoints

GET

Returns the current image based on the key's rotation interval.

{
  "index": 2,
  "total": 10,
  "filename": "photo-1712345678.jpg",
  "url": "https://your-server/uploads/photo-1712345678.jpg",
  "interval_minutes": 5,
  "next_at": "2026-04-12T11:05:00.000Z",
  "next_in_ms": 42000
}
GET

Returns all uploaded images as an array.

[
  { "index": 0, "filename": "cat-1712345678.jpg", "url": "https://your-server/uploads/cat-1712345678.jpg" },
  { "index": 1, "filename": "dog-1712345679.jpg", "url": "https://your-server/uploads/dog-1712345679.jpg" }
]

Example (curl)

curl -H "x-api-key: YOUR_KEY" https://your-server/api/slideshow/current
